# 图标规范

> 跨课件通用。浙商银行模板图标结构定义。

## 1. 图标结构

- **尺寸**：540 × 540 EMU 的 GROUP 形状
- **组成**：每个图标由 2 个子元素构成
  1. 椭圆（底色圆形背景，NoFill 状态 = 可着色）
  2. Freeform/Group（矢量图形本体，用 theme_color 填充）

## 2. 关键特性：theme_color 引用

图标本体用 **theme_color 引用**，**改主题色即全局换色**——无需逐个修改。

## 3. 图标分类（10 类）

- 人物类
- 办公用品
- 信息图标
- 金融类
- 行为类
- 管理类
- 常用符号
- 文档管理
- （其他两类待补）

## 4. 使用规范

- 使用时统一缩放到正文行高（≈ 1.2em）
- 不与正文行距冲突（保持 1.3 行距不变）
- 颜色跟随主题色，不要手动指定 srgbClr

## 5. 导出规范

- 格式：**SVG**（首选）/ **EMF**（备用）
- 工具：LibreOffice headless / PowerPoint VBA
- python-pptx **不支持**直接导出图片，需绕道
