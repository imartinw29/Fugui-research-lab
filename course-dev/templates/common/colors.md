# 配色方案（浙商银行主题）

> 跨课件通用。生成任何浙商银行相关课件时严格遵守。

## 1. 主色板

| 角色 | 色值 | 用途 |
|------|------|------|
| dk1 / accent6 | #2C2A29 | 正文文字、深色背景 |
| lt1 | #FFFFFF | 白色底 |
| **accent1** | #9B0000 | **主色（深红）· 章扉页主标题、关键数字** |
| **accent2** | #C7000B | **辅助色（亮红）· 重点强调** |
| accent3 | #666464 | 灰色 · 正文次要文字 |
| accent4 | #BFC0C0 | 浅灰 · 装饰底纹、边框 |
| **accent5** | #E60012 | **强调红 · 单一关键柱/扇区/标记** |

## 2. 配色使用三原则

- **对比色**：用于强调（accent5 vs lt1 白色）
- **渐变色（递进关系）**：accent1 → accent2 → accent5，**仅用于时间序列/数据大小**
- **多色并列**：accent1 / accent2 / accent3 / accent4 并列，**最多四种**

## 3. 信息图表硬规则（红黑白配色陷阱）

红黑白配色无法靠颜色区分"上涨/下跌/中性"，强制三件套：

1. **数据标签直接打在柱顶/扇区外**——不靠图例区分
2. **关键柱/扇区用 accent5 实底**——其余用 accent3 浅灰
3. **标题就是结论**（如"X 月销售环比下降 12%"），不是"销售趋势图"

## 4. 反面案例 vs 正确示范

### ❌ 反例：靠颜色区分
饼图四块扇区分别用 accent1/2/3/4 填色，无数据标签——区分不了谁是谁

### ✅ 正例：靠标签+位置
饼图扇区统一 accent3 浅灰，只有"重点扇区"用 accent5，外侧引线标名称+数值
