# 教学 PPT 备注栏模板

> 仅适用于**教学课件**（培训、内训、参赛课件）。
> 汇报类 PPT 见 `../reporting/notes-template.md`。

## 五要素（强制）

每页备注必须包含以下五部分，缺一项视为不完整：

1. **【教学目标】** 动宾结构（不用"了解/掌握"等模糊词）
   - 例：能在 5 分钟内识别客户的三类隐性投诉信号
   - 反例：了解客户投诉的类型

2. **【教学方法】** 从以下选：
   - 讲授
   - 互动提问
   - 案例研讨
   - 情景模拟
   - 实操模拟
   - 小组研讨
   - 练习

3. **【时间分配】** XX 分钟（精确到分钟）

4. **【讲解话术】** 详细话术 + 上下页过渡语

5. **【注意事项】** 可能出问题及应对策略

## 模板实例

```
【教学目标】能在 3 分钟内说出化解客户不满的四步流程
【教学方法】讲授 + 情景模拟
【时间分配】3 分钟
【讲解话术】
   引入："同学们，我问大家一个问题——当你买理财亏了 1 万块，
   走进银行第一句话会说什么？"
   （停顿 5 秒）
   "对，先骂人。那如果你是大堂经理，面对这个客户，
   你怎么办？今天我们讲四步化解法。"
   过渡语："接下来我们看第二步——转移场景。"
【注意事项】
   - 学员可能会问"如果客户情绪很激动怎么办"——预案：提前准备话术"先让他宣泄完再讲道理"
   - 时间紧时跳过案例研讨环节
```

## 评委隐藏评分点

比赛时评委翻 PPT 的"备注页"很关键——他们看的不是页面，是**教学话术完不完整、教学方法选的对不对、应急方案有没有**。备注栏不完整直接扣分。
