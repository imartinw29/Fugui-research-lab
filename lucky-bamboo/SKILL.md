## 策略参数

```
布林带: BOLL-M(20), ±2σ
KDJ:    (14, 3, 3)
MACD:   (12, 26, 9)
周期:   日线

买入信号 = 收盘价 ≤ 布林中轨 AND K<30 AND J<20 AND MACD绿色柱状线开始收窄
卖出信号 = MACD死叉（DIF下穿DEA）
```

**为什么用MACD死叉卖出而非碰轨卖出：** 碰轨在趋势行情中会踏空主升浪。MACD死叉 = 确认趋势反转才离场，"让利润奔跑"。

## 筹码峰（CYQ）辅助指标

通过 DFCF 妙想 API 查询，返回四个关键指标：

| 指标 | 含义 |
|------|------|
| 平均持仓成本 | 所有持股人的平均买入价（元） |
| 获利盘比例 | 当前价之上有多少持仓在盈利（%） |
| 70%筹码集中度 | 含70%筹码的价格区间宽度（%） |
| 90%筹码集中度 | 含90%筹码的价格区间宽度（%） |

**与四灯的配合：**
```
四灯全亮 + 当前价 < 均价 → 🟢 双重共振（最强信号）
四灯全亮 + 当前价 > 均价 → 🟡 获利盘有兑现压力
四灯不全 + 当前价 < 均价 → 🟡 筹码低位但时机未到
四灯不全 + 当前价 > 均价 → 🔴 都不满足
```

🔴 **策略适用性：** 回测结果基于历史数据，不保证未来表现。实盘前确认：当前市场环境是否匹配策略假设？仓位是否在承受范围内？

## MACD 八形态自动检测（v2.8）

`quick_scan.py` 自动识别八种经典MACD形态，与四灯互补——**四灯找买入时机，形态判断趋势结构。**

| 分组 | 形态 | 核心逻辑 |
|------|------|----------|
| 零轴下二次金叉 | 小鸭出水 / 海底捞月 | 零轴下金叉→死叉→再金叉，空头衰竭 |
| 零轴上回踩不破 | 空中缆车 / 漫步青云 | 零轴上死叉→DIF不破零轴→再金叉，洗盘结束 |
| 零轴上佛手 | 佛手向上 | 金叉→DIF回踩未死叉→抬头，强势延续 |
| 零轴下天鹅 | 天鹅展翅 | 金叉→DIF回踩未死叉→抬头，弱势转强 |
| 穿零轴粘合 | 空中缆绳 | 零轴下金叉→上穿→粘合→分离抬头 |
| 零轴下粘合 | 海底电缆 | 零轴下长期粘合→金叉，底部蓄势 |

详见 `references/macd-8-patterns.md`

## 尾盘选股策略

**核心理念：** 不看题材板块，只看指标——14:50扫描，符合条件次日开盘卖。胜率略高于抛硬币，真正的优势在赔率。

**两步法：**
1. **海选层** — 全A股：涨幅2-9% + 换手>3% + 量比>1.5 + 去ST
2. **验证层** — 逐只拉1分钟分时，三维评分（0-100）：
   - 均价线上方时间占比（40分）
   - 下午走势强度（30分）
   - 尾盘资金托举（30分）

**纪律：**
- ⏰ 只在14:50跑
- 🚫 不合格就空仓
- 📉 大盘共振下跌 → 空仓日
- 📈 集体涨停 → 空仓日

**数据源：** 东方财富公开API（`push2his.eastmoney.com`，HTTPS + urllib，无需密钥）
**脚本：** `python scripts/tail_market_scanner.py --top 20 --debug`

## 投资框架

### 基础设施型垄断
单价极低但不可或缺，占成品成本<1%，缺了产线停摆。认证周期1-3年，客户不敢换→天然定价权。

### 卖耗材 > 卖铲子
设备卖一次吃十年，耗材用了就得一直买。详见 `references/investment-framework.md`

### 适用边界

- ✅ 几百亿~几千亿市值的高弹性标的
- ✅ BB中下轨+KDJ超卖+MACD力竭
- ✅ 有不可替代性的业绩股（认证+工艺+客户惯性）
- ❌ 万亿级大盘股（弹性不足）
- ❌ 纯概念/辟谣股（一字涨跌停跑不掉）
- ❌ 单边下跌趋势（等MACD拐头）

### 策略局限性

| 风险 | 表现 |
|------|------|
| 市场风格切换 | 单边下跌中BOLL中轨一路下移，每次抄在中轨都被埋 |
| MACD延迟确认 | 死叉出现时已回吐大部分利润 |
| 标的市值膨胀 | 低位的信号质量≠高位的信号质量 |
| 行业不匹配 | 参数在特定行业优化，其他行业未必适用 |

**核心原则：** 策略告诉你概率，不是保证。价值在于长期胜率和盈亏比，不在于单笔对错。

### BOLL收口（Squeeze）

- **收口+四灯+价格贴下轨**：弹簧压紧，突破往上概率大
- **张口+四灯+价格在下轨**：波动放大，方向不明，等收口
- **收敛末端**：放量阳线=入场确认，跌破下轨=止损信号

## 网络与降级

### 网络坑
- WSL `requests` 库连 `push2his` 会 `RemoteDisconnected`，`urllib.request` 直连HTTPS可通
- Python urllib TLS问题：需 `ssl._create_unverified_context()`
- 诊断：`curl -s --max-time 5 "https://push2his.eastmoney.com/api/qt/stock/kline/get?secid=1.688008&..."`

### 降级链

1. `quick_scan.py`（push2his，完整 OHLC → KDJ+BOLL+MACD）→ 首选
2. curl能通但quick_scan报error 102 → exec()直调 `scan()` 函数
3. `fallback_scan.py`（妙想收盘价 → numpy算BOLL/MACD，KDJ估算）
4. 裸调妙想API分三次查收/高/低价
5. 新浪日K API（最终降级，完整OHLC但无换手率）

### 数据源故障

| 触发条件 | 修复 | 兜底 |
|------|------|------|
| push2his宕机 | curl存活检测 → fallback_scan | 妙想分三次查 |
| A/H股混淆 | 跳过「港」后缀，优先A股字段 | 手动输入 |
| dfcf API无响应 | 重试1次间隔3秒 | 标注"数据不可用" |
