# 投资框架 — 富贵竹

## 基础设施型垄断

### 定义
一种商业模式：产品单价极低（几分~几毛），占客户成品总成本<1%，但缺了整条产线停摆。客户更换供应商的认证周期1-3年，一旦认证通过形成事实垄断。

### 框架类比

| 类比 | A股标的 | 全球竞争者 | 垄断来源 |
|------|------|:---:|------|
| YKK拉链 | — | 1家 | 品质+产能+客户惯性 |
| 内存接口芯片 | 澜起科技 688008 | 3家 | 认证周期+专利 |
| MLCC成品 | 风华高科 000636 | 4家 | 材料配方+叠层工艺+车规认证 |
| MLCC钛酸钡 | 国瓷材料 300285 | 3家 | 纳米级粉体工艺 |

### 定价权来源
- 涨价50%，客户成本只增加0.5%，不在乎
- 断供一天，客户整条产线报废，赌不起
- 所以这类公司毛利率通常40%+，周期底部也不亏

## 三层过滤体系

```
第一层：基本面过滤
  ↓ 有不可替代性？（认证壁垒+工艺壁垒+客户惯性）
  ↓ 有持续业绩？（非概念炒作，非辟谣股）
  ↓ 市值适中？（几百亿~几千亿）

第二层：技术面执行
  ↓ 买入 = 收盘≤BB中轨 & K<30 & J<20 & MACD柱收窄
  ↓ 卖出 = MACD死叉（DIF下穿DEA）
  ↓ 止损 = 买入后跌5-8%强制平仓

第三层：仓位管理
  ↓ MACD多头时持仓，死叉时空仓
  ↓ 买入信号连发时，第2-3天确认入场
  ↓ 回测过的标的 → 凯利公式算仓位；没回测的 → 最多1手
```

### 仓位管理细则

#### 凯利公式（Kelly Criterion）

```
f = (p × b - q) / b

p = 胜率（策略历史信号胜率 ≈ 60%，基于回测4/4信号）
b = 盈亏比（预期盈利 / 预期亏损，本策略取 2:1 = 止盈+10% / 止损-5%）
q = 1 - p
```

**本策略的凯利值：** f = (0.6×2 - 0.4) / 2 = **40%**

实操含义：总资金的40%可以压在一个信号上。但——

**凯利公式只适用于有回测数据的标的。** 没有胜率和赔率输入，凯利无法计算。

#### 头寸公式（以损定量）

凯利告诉你押多少比例，头寸公式把它翻译成具体手数：

```
手数 = 总资金 × 单笔风险% / (买入价 - 止损价) / 每手股数

单笔风险% = 总资金的 1-2%（保守）或凯利算出的比例（激进）
买入价    = 实际入场价
止损价    = 买入价 × (1 - 止损幅度%)
```

**实操示例（澜起科技，总资金8万）：**

```
买入价 = 240   止损幅度 = 5%   止损价 = 228   单笔风险 = 2%
手数 = 80000 × 2% / (240 - 228) / 100 = 1600 / 12 / 100 = 1.33 → 1手
```

如果单笔风险取凯利值40%：
```
手数 = 80000 × 40% / 12 / 100 = 32000 / 12 / 100 = 26.7手 → 远超总资金
```

**这就是小资金用凯利的现实：** 凯利40%在8万里是3.2万，但止损距离12块太近，算出来的手数要么只能1手（保守），要么买不起（凯利全押）。结论：**集中持仓1-2手就是答案，不用纠结公式算出来的小数。**

#### 止损规则（强制）

| 规则 | 参数 | 说明 |
|------|------|------|
| 硬止损 | 买入价 -5% | 无条件执行，不找理由 |
| 宽止损 | 买入价 -8% | 仅限高波动标的（科创板/创业板），给假突破留缓冲 |
| 时间止损 | 持仓15天未触发卖出信号 | 资金占用成本过高，主动离场 |
| 周五不开仓 | 周四/五出现信号 → 等周一 | 周末两天黑天鹅窗口 |

止损不是认输，是**保留下一局的筹码**。

#### 仓位分级规则

| 类型 | 条件 | 仓位上限 | 凯利输入 |
|------|------|:--:|------|
| 🟢 **主力仓位** | 回测过 + 四灯+筹码共振 | 凯利公式算的手数 | 有胜率有赔率 |
| 🟡 **观察仓** | 回测过 + 信号不全 | 0（等待） | — |
| 🔴 **试探仓** | 未回测 / 策略外标的 | **1手** | 没法算 |
| ⚫ **娱乐仓** | 纯概念/催化剂赌注 | 1手以下 | 没法算 |

**试探仓的铁律：** 未经回测验证的标的，不管你多看好，初始仓位只许买1手。卓胜微买300股=3手=违反了这条规则。

#### 小资金的凯利现实

总资金<10万时，凯利算出来的金额可能只够买1-2手。这不是公式的问题——是你的资金量本来就只适合**集中持仓**。同时持仓不超过2只回测标的，每只1-2手，就是小资金的最优解。

凯利不是让你「买更多」，是让你「买多少心里有数」。
