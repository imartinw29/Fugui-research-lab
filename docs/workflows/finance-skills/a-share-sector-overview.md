# A股行业深度分析（Sector Overview）

> **目标：** 不是泛泛的行业介绍，而是能支撑交易决策的结构化分析。

## 适用场景

- 行业轮动 / 主题投资的选赛道
- 新行业研究的入门框架
- 投资委员会行业配置提案
- 作为 `a-share-comps-analysis` 的前置步骤（先定赛道，再挑标的）

---

## ⚠️ 数据源优先级

1. **东方财富 API**（`dfcf-finance-tool`）— 板块行情、个股财务、资金流向
2. **巨潮资讯网** / 公司公告 — 监管政策、行业新闻
3. **Web 搜索** — 仅补充行业新闻和研报观点
4. **绝不用**未经验证的 AI 生成预测数据

---

## 工作流（6 步）

### Step 1：界定范围

- **行业：** 精确到申万二级或三级行业（如"半导体—数字芯片设计"而非"科技"）
- **目的：** 投资研判 / 研报输出 / 内部研究
- **深度：** 概览（5-8 页 PPT）或深度（15-25 页 PPT）
- **角度：** 中性全景 vs. 主题驱动（如"AI 算力链国产替代"）

### Step 2：市场规模与增长

**市场规模：**
- 行业整体规模（亿元）及出处（券商研报/行业协会/招股书）
- 近 3 年增速（CAGR）
- 未来 2-3 年增速预测及核心假设
- A 股该板块总市值、上市公司数量

**产业链图谱：**
```
上游（原材料/设备）→ 中游（制造/服务）→ 下游（应用/客户）
```
标注每个环节的代表性 A 股公司。指出**利润最厚的环节在哪里**。

### Step 3：行业结构与竞争

**集中度：**
- 行业 CR5 / CR10（营收或市值占比）
- 是"大而集中"还是"小而分散"？
- 龙头公司市值与行业总市值的比值

**竞争维度：**
- A 股特有的竞争要素：牌照壁垒、地方保护、融资能力、产能扩张节奏
- 价格战风险：行业毛利率趋势
- 新进入者威胁：IPO/定增扩产潮

**商业模式分类：**
- 行业内存在哪些不同商业模式？（toB/toC/toG、轻资产/重资产、产品/服务/平台）
- 哪种模式 ROE 最高？

### Step 4：A 股特色因子

以下是 Anthropic 原版没有、对 A 股至关重要的分析维度：

**政策与监管：**
- 近期监管政策梳理（利好/利空/中性）
- 产业政策支持力度（国家级/省级/补贴规模）
- "政策底"与"市场底"的判断

**资金面：**
- 北向资金对该板块的持仓变化（增持/减持趋势）
- 融资余额趋势
- 机构持仓占比变化（公募/私募/保险）

**情绪与估值水位：**
- 板块 PE/PB 当前值 vs. 历史分位（5 年）
- 换手率趋势（过热/冷清信号）
- 板块年内涨跌幅 vs. 沪深 300

### Step 5：投资含义

**机会识别：**
- 行业中最具性价比的子赛道
- 赔率 vs. 胜率的权衡
- 催化剂事件（政策落地、新产品发布、产能释放、业绩拐点）

**风险提示：**
- 行业核心分歧（多空双方各自的论据）
- 可能颠覆行业叙事的变化
- 估值过高的警示信号

**A 股实战清单：**
- 当前值得关注的 3-5 只标的（简要逻辑）
- 什么情况下应该离场
- 观察的时间窗口（季报季、政策窗口、年报季）

### Step 6：输出格式

**基础输出（Markdown 分析报告）：**
```
# [行业名称] 行业分析
**日期：** YYYY-MM-DD
**分析师声明：** 本报告基于公开数据，不构成投资建议

## 1. 行业全景
- 规模、增速、产业链

## 2. 竞争格局
- 集中度 + 龙头分析 + 商业模式

## 3. A 股维度
- 政策 + 资金 + 估值水位

## 4. 投资含义
- 机会 + 风险 + 观察清单

## 5. 数据附表
- 板块核心标的列表（代码/市值/PE/ROE/营收增速）
```

**进阶输出（联动 `powerpoint` skill 出 PPT）：**
- 使用 `a-share-comps-analysis` 先跑可比分析
- 将行业分析 + 可比分析数据合流到 PPT（见下方整合工作流）

## 与 Anthropic 原版的差异

| 维度 | Anthropic 原版 | A 股版 |
|------|---------------|--------|
| 行业分类 | GICS | 申万 |
| 数据源 | Morningstar/S&P MCP | 东方财富 API |
| 估值锚 | EV/EBITDA | PE/PB + 历史分位 |
| 特色分析 | 无 | 政策因子、北向资金、融资情绪 |
| 货币 | USD/Billions | 亿元 |
