# Fugui Research Lab

> 一个持续沉淀方法论、研究框架、智能体工作流与实践笔记的个人研究实验室。

这里不追热点，不做 AI slop，也不把“多写一点”误当成“研究更深”。

我更关心的是另一类东西：那些**经得起回看、能够重复使用、可以逐步迭代**的方法与结构。

这个实验室目前主要记录四类内容：

## Methods

研究方法与分析框架。

比如：

- 横纵分析法
- 深度研究的结构化拆解
- 问题定义与变量识别

## Workflows

可执行的工作流、SOP 与智能体协作方法。

比如：

- Deep Research workflow
- 文档提取与整理流程
- Agent 工作流设计

## Notes

值得留下来的判断、短文与工作笔记。

它们不一定完整，但应该有可复用的部分。

## About this lab

这不是内容农场，也不是个人情绪回收站。

更像一个长期维护的认知工坊：

- 把零散经验整理成结构
- 把 prompt 提升成 skill
- 把一次性的解决方案沉淀成长期资产

如果某篇内容让你少走一点弯路，那它就值回这一页纸。
