# Fugui Research Lab

A personal research lab for methods, workflows, and durable thinking.

## What this repo is for

This repository hosts a lightweight documentation site for sharing:

- research methods
- decision frameworks
- agent workflows
- reusable skills and SOPs
- practical notes worth revisiting

The goal is simple: publish things that stay useful after the hype cycle passes.

## Local preview

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
mkdocs serve
```

## Deploy

This repo is configured for GitHub Pages via GitHub Actions.
